# share/共有ドライブ ディレクトリ構成

## 全体構成

```text
share/共有ドライブ/
├─ プロジェクト/
│  ├─ 京橋信用ソリューションズ株式会社/
│  ├─ 医療法人社団 恒一会 かえで総合病院/
│  ├─ 医療法人社団 蒼樹会 みなみ野女性医療センター/
│  ├─ 医療法人社団 蒼泉会 ひがし丘総合病院/
│  ├─ 株式会社東都人材プラットフォーム/
│  ├─ 株式会社青嶺不動産アセットマネジメント/
│  ├─ 株式会社青潮モビリティサービス/
│  ├─ 株式会社青葉バイオメディカル機器/
│  ├─ 白峰信用リスク評価株式会社/
│  └─ 青葉与信マネジメント株式会社/
└─ 社内管理/
   ├─ データアステル社内管理_決裁基準.md
   ├─ データアステル社内規定_パスワード導出規則.docx
   ├─ 座席表.pptx
   └─ 社内用語集.docx
```

## プロジェクト配下の共通構成

各プロジェクト会社の基本構成は以下です。

```text
<会社名>/
├─ 00.提案/
├─ 01.契約/
├─ 02.計画/
├─ 03.データ/
├─ 04.分析/
│  ├─ analysis_outputs/
│  │  └─ experiments/
│  └─ analysis_project/
│     ├─ configs/
│     ├─ data/
│     ├─ notebooks/
│     ├─ reports/
│     │  └─ figures/
│     ├─ scripts/
│     └─ src/
├─ 05.会議/
│  ├─ 会議録/
│  └─ 報告資料/
└─ 06.報告書/
```

## 例外・差分

一部プロジェクトでは、共通構成に加えて以下のフォルダがあります。

```text
株式会社青嶺不動産アセットマネジメント/
└─ 00.提案/
   └─ old/

医療法人社団 恒一会 かえで総合病院/
└─ 04.分析/analysis_project/src/__pycache__/

青葉与信マネジメント株式会社/
├─ 04.分析/analysis_project/
│  ├─ artifacts/
│  │  └─ analysis_outputs/
│  └─ src/__pycache__/
└─ 06.報告書/
   └─ old/
```

また、`株式会社青葉バイオメディカル機器` の `04.分析/analysis_project/` には、確認時点で `reports/figures/` がありませんでした。

```text
株式会社青葉バイオメディカル機器/
└─ 04.分析/analysis_project/
   ├─ configs/
   ├─ data/
   ├─ notebooks/
   ├─ scripts/
   └─ src/
```
